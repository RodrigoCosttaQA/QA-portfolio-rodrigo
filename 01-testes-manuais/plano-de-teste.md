
Plano de Teste – Swag Labs
1. Objetivo

Validar as principais funcionalidades do sistema Swag Labs, garantindo que o fluxo de login, listagem de produtos, carrinho e logout estejam funcionando corretamente.

2. Escopo
Funcionalidades que serão testadas:

Login válido

Login inválido

Listagem de produtos

Adição ao carrinho

Remoção do carrinho

Finalização de compra

Logout

3. Tipo de Teste

Teste Funcional

Teste Manual

Teste Exploratório

Teste de Regressão (conceitual)

4. Ambiente de Teste

Navegador: Google Chrome

Sistema Web

URL: https://www.saucedemo.com/

5. Critérios de Aceite

O sistema deve permitir login com credenciais válidas.

O sistema deve bloquear login inválido.

Produtos devem ser exibidos corretamente.

O carrinho deve atualizar corretamente ao adicionar/remover itens.

O checkout deve finalizar corretamente.

6. Riscos

Instabilidade do site de demonstração.

Diferença de comportamento entre navegadores.
